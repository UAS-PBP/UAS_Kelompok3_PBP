HomeLinkID adalah website katalog rumah dimana tempat untuk berinteraksi antara perusahaan dan pembeli. Website yang kami punya memiliki keunikan dimana orang yang dapat mengupload rumah disini adalah perusahaan atau PT yang sudah bersertifikat sehingga para pembeli dapat merasa aman untuk membeli rumah. 

Website ini memiliki benefit tersendiri baik bagi para pembeli maupun perusahaan, dimana pembeli dapat merasa aman sekaligus membeli rumah sesuai dengan budget dan wilayah yang diinginkannya dan dapat membeli secara langsung ke perusahaan tersebut. Perusahaan juga memiliki benefit nya yaitu perusahaan dapat menjual, menjualnya dengan langsung sekaligus dapat mengorganisir siapa saja yang ingin survey atau siapa saja agen yang ditugaskan kedalam rumah. Sehingga Website HomeLinkId ini tidak hanya berguna bagi satu sisi saja namun juga berguna untuk kedua sisi yaitu Perusahaan dan Pembeli.

Fitur-Fitur Utama yang dimiliki weebsite ini adalah:
Dari sisi Buyer:
1. Search Rumah sesuai dengan Budget dan Wilayahnya
2. Search Rumah yang sedang Promo sesuai dengan budget dan wilayahnya
3. Reservasi Survey ke Rumah yang ingin dikunjungi dan berinteraksi secara langsung ke agen / perusahaan
4. Chatbot yang dimana pembeli dapat menanyakan seputar rumah atau seputar preferensi yang ingin dibeli

Dari sisi Perusahaan
1. Darshboard rumah yang merekap semua yang diinput
2. Input rumah yang otomatis menghubungkan ke dalam home si pembeli
3. Jadwal reservasi yang berasal dari input reservasi survey pembeli
4. Input Agen agar memudahkan untuk menghubungi

Cara Menjalankan website:
1. Clone repository  Laravel kedalam folder yang dimiliki
3. Ketik Composer Install dan npm install dan tunggu beberapa menit
4. Copy .env-example menjadi .env sesuai database yang dimiliki
5. Ketik npm run dev di satu terminal sekaligus terminal lain ketik php artisan serve
6. Ctrl + Klik link yang ada di php artisan serve
